### 🗳️ Vote Management System

A simple Voting Management System built using Python, Object-Oriented Programming (OOP), and Streamlit.
This application allows users to vote for candidates, prevents duplicate voting, and displays the voting results with the winner.

##🚀 Features

Simple voting interface

Prevents duplicate voting

Displays vote count for each candidate

Shows the winning candidate

Built using Python OOP

##🛠️ Technologies Used

Python

Streamlit

Object-Oriented Programming (OOP)

📊 Application Workflow

User enters Voter ID

Selects a Candidate

Submits vote

System records the vote

Prevents duplicate voting

Displays results and winner
